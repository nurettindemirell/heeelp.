# heeelp.

heeelp. is a simple student web application for English-Turkish language support.

is active now: https://heeelp-eta.vercel.app/

It has three main functions:

- **Sentence Control**
  Checks whether an English sentence is correct or incorrect.
  If the sentence is correct, it suggests an alternative.
  If it is incorrect, it gives a corrected version and an alternative.

- **ENG to TUR**
  Translates English text into Turkish.

- **TUR to ENG**
  Translates Turkish text into English.

## Purpose of the Project

This project was created as a student project to practice:

- building a frontend and backend together
- using API routes in Next.js
- working with AI-based text processing
- designing a responsive web interface
- creating a simple but useful language tool

## Technologies Used

- **Next.js**
- **JavaScript**
- **CSS**
- **OpenAI API**

## Features

- English sentence checking
- English to Turkish translation
- Turkish to English translation
- chat-style interface
- light and dark mode
- responsive design for desktop and mobile
- simple custom UI styling
